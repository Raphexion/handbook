Building
========

This chapter describes how to build something with OE-lite, fx. how to build a
specific OE-lite recipe, a Linux kernel image, a JFFS2 root filesystem image,
an SDK toolchain image, and so on.

Building is done with the OE-lite Bakery sub-command called "bake".
